# Todo In-Memory Python Console App

This is Phase I of the Hackathon project, focusing on building a basic command-line todo application.

## Features
-   Add tasks with a title and description.
-   List all tasks with their status.
-   Update existing task details.
-   Delete tasks by ID.
-   Mark tasks as complete or incomplete.

## Setup Instructions
1.  **Navigate to the project directory:**
    ```bash
    cd "hackthon panaversity phase 1/todo-app"
    ```
2.  **Install dependencies (if any):**
    ```bash
    # Currently no external dependencies, but if there were, you'd install them here.
    # For example: pip install -r requirements.txt
    ```
3.  **Run the application:**
    ```bash
    python src/main.py
    ```

## Usage
Follow the on-screen prompts to interact with the todo application.
